Shopping List App
Простое Android-приложение для управления списком покупок.
Позволяет добавлять товары, отслеживать общую сумму и управлять списком.

Функционал:
Подсчёт общей суммы всех добавленных товаров
Отображение введённых данных в виде списка
Добавление новых элементов (название и цена)
Удаление элемента с помощью свайпа (влево или вправо)

Как работает приложение:
Пользователь вводит название товара и его цену
Элемент добавляется в список
Приложение автоматически пересчитывает общую сумму
Все элементы отображаются в виде списка
Для удаления элемента достаточно выполнить свайп влево или вправо
